1. D'abord, nous devons cloner le référentiel :

```bash
git clone https://github.com/nkanaev/yarr
```

```bash
docker build -t yarr .
```

```bash
docker run -d \
  -p 7070:7070 \
  -v $(pwd):/data/ \
  --restart unless-stopped \
  yarr
```
